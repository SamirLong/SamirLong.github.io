# 1.先用普通用户登录
浏览器输入`192.168.1.1`
输入光猫背面的账号密码
然后新建一个窗口访问`http://192.168.1.1/webcmcc/telnet.html`
会显示open telnet success!

# 2.用命令进入telnet
打开控制面板点击程序点击启动或关闭windows
打开telnet确认保存

<img width="413" height="386" alt="Image" src="https://github.com/user-attachments/assets/338ddc06-b701-45b6-b564-f2be324d6553" />

win+r输入cmd打开命令行
输入`telnet 192.168.1.1`

<img width="335" height="205" alt="Image" src="https://github.com/user-attachments/assets/08463624-dfdc-4a14-8cbc-3d12f4c95a4d" />

输入你的光猫背面的账号
再输入密码（密码不显示）

<img width="342" height="209" alt="Image" src="https://github.com/user-attachments/assets/50cc33ea-4c41-4d6f-8a2c-4b754a83c5df" />

再次输入`cd /config/work`

# 3.查询自己的宽带拨号上网的账号密码
下载中国移动app
搜索宽带 找到宽带密码重置
重置密码
记下宽带账号和密码留着备用

# 4.将光猫改桥接模式
进入光猫后台输入管理员账号密码
点击网络-宽带设置
找到 INTERNET_B_VID_41
截图保存参数
将连接模式改成bridge保存即可

<img width="965" height="845" alt="Image" src="https://github.com/user-attachments/assets/d7fab785-3586-489a-8a9a-55b47819b993" />

# 5.如果无法修改则需要新建一个（可以修改的话这一步省略）
点击新建
连接模式选择bridge
VLAN ID :保持一致保存即可
然后删除之前的INTERNET_B_VID_41即可
# 6.设置路由器拨号上网
浏览器输入路由器管理地址
点击上网设置
模式选择宽带拨号上网
输入用户名密码即可
